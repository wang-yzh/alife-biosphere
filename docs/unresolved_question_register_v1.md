# Unresolved Question Register v1

## Purpose

This is the first unified unresolved-question register for `alife_biosphere`.

It gathers the main open questions that were previously scattered across the
library and assigns them simple status labels:

- `active_now`
- `postponed`
- `out_of_scope_for_now`

## Active Now

| ID | Question | Why it is still open | Next evidence that would help |
| --- | --- | --- | --- |
| `UQ-01` | Which cheap interaction-network summaries best approximate connectedness in our small worlds? | Connectedness is still a proxy family, not a settled minimal metric stack. | Compare candidate summaries against rescue brittleness and habitat turnover. |
| `UQ-02` | How should archive/source trust update differ from direct partner trust? | The channels are now separated conceptually, but not yet in a settled update rule. | Controlled probes with direct trust only, source trust only, and split-channel trust. |
| `UQ-03` | When should a local release/reorganization cycle count as a true regime shift? | Adaptive-cycle behavior and regime-shift interpretation still overlap. | Compare repeated cycles against persistent post-disturbance state change. |
| `UQ-04` | How much cultural visibility is enough before archive overload dominates? | Overload is real in theory, but the useful operating window is unknown here. | Archive visibility ablations under stable and changing habitat conditions. |
| `UQ-05` | What level of group asymmetry should count as evidence for meaningful higher-level organization? | The project has a claim standard, but not a practical ecological threshold. | Long-run comparison of symmetric versus asymmetric groups under coordination or damage stress. |
| `UQ-06` | How should rescue sources be classified reliably from logs? | Rescue-source classification is required in design but not yet operationally reliable. | Multi-source disturbance probes with known interventions and retrospective checks. |
| `UQ-07` | Which archive metadata fields are indispensable in practice, and which are only nice to have? | Provenance and domain tags are justified, but the minimum viable metadata set is unknown. | Compare full metadata against reduced metadata in archive-use probes. |
| `UQ-08` | How should lineage persistence be separated from open-ended innovation in routine summaries? | The project knows these are different, but routine summaries still do not separate them cleanly. | Side-by-side summaries of novelty, persistence, and structural change across long runs. |

## Postponed

| ID | Question | Why it is postponed | What would reactivate it |
| --- | --- | --- | --- |
| `UQ-09` | How should adaptive-cycle phase be quantified beyond proxy profiles? | Proxy profiles are good enough for now; stronger phase estimation is not yet the main bottleneck. | Once longer runs exist with multiple cycle-like trajectories. |
| `UQ-10` | How far should signaling move toward richer conventions or code drift? | Low-bandwidth signaling still needs to become real before richer convention dynamics matter. | After minimal signaling and trust probes exist. |
| `UQ-11` | When should symbiosis or mutualistic partner choice become first-class? | Theoretical groundwork exists, but current implementation is still upstream of stable antagonism and group persistence. | Once parasites, defense, and local coexistence are observable. |
| `UQ-12` | How much embodied morphology should the first world eventually include? | The substrate is still too early for morphology to be on the critical path. | If ecological roles become too abstract to study further without body structure. |

## Out Of Scope For Now

| ID | Question | Why it is out of scope for now |
| --- | --- | --- |
| `UQ-13` | Can this project demonstrate strong biological open-ended evolution? | The evidence base and implementation state are still too early for such a claim. |
| `UQ-14` | Can this project demonstrate general intelligence? | That is not a near-term target and would distort the research priorities. |
| `UQ-15` | Can the first version support realistic economic or financial ecology? | The current line is focused on world evolution, not market realism. |
