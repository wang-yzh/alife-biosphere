# Unified Bibliography v1

## Purpose

This is the first unified working bibliography for `alife_biosphere`.

It is not a full dump of every paper link that appears across the topic
reviews. It is a curated cross-topic index of the papers that most directly
anchor the current research position.

Role labels:

- `foundation`
- `mechanism`
- `measurement`
- `caution`
